Ground plane shadows demo -- draws a teapot and projects
its shadow onto 3 axis-aligned planes. Click and drag with
the mouse to move the light around. Right-click for a menu
of options. If you're running on a slow system then toggle
the teapot off -- the cube runs much faster.

Demo files:
-----------
shadows.exe     - demo executable
glut32.dll      - GLUT DLL required to run demo

main.c          - shadow projection code and demo

shadows.dsw     - Visual C++ workspace
shadows.dsp     - Visual C++ project
